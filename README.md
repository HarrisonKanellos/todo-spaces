# Spaces

A todo list app that supports creation of todo tasks within 'spaces' that act as seperate todolists. Data is stored using `localStorage` for persistence.

By deafault users are provided a 'Common' space, it's purpose is to be used as a list for general or miscellaneous tasks. The 'Completed' list displays tasks from all spaces that are marked as completed.

The app has a minimalist and clean colour scheme and user interface, providing focus and simplicity to users.
